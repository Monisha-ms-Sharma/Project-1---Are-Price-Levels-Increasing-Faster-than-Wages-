# Data Directory

This folder contains all datasets used throughout the project.

## raw/

Original source files.

These files are never modified.

### excel/

Contains the original BLS Excel workbooks downloaded from the U.S. Bureau of Labor Statistics.

### api/

Contains datasets downloaded automatically from the BLS API.

### csv/

Contains CSV files converted from the original Excel files.

## interim/

Contains cleaned datasets used during preprocessing.

## processed/

Contains analysis-ready datasets generated after feature engineering.

## external/

Contains optional datasets from other public sources (FRED, Census, BEA, etc.).
